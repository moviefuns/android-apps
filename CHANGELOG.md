# Changelog

All notable releases across the Barburas Android Apps suite are documented here. Each app also maintains its own 'CHANGELOG.md' with detailed per-version notes.

---

## Nóta — Nextcloud Notes Client ('com.brbrs.nota')

### [1.0.2] - 2026-06-09
- Handwritten logo in notes list, app lock screen, and login screen
- GitHub link in Settings → Support section
- Tasks.org integration
- Forest green accent color matching new icon
- New app icon
- Image markdown hidden in editor and list preview
- Full-size images in single note view
- Various bug fixes and light mode improvements

### [1.0.1] - 2026-06-03
- Dark and light mode with instant toggle
- Category dropdown picker
- Share text and images from other apps
- Authenticated image rendering from Nextcloud
- Image thumbnails in note list cards
- Support section in Settings
- Date format updated to YYYY/MM/DD | HH:MM

### [1.0.0] - 2026-05-31
- Initial release
- Nextcloud Login Flow v2 authentication
- Offline-first sync
- Markdown editor with preview
- Biometric app lock and per-note lock
- Category filtering and search
